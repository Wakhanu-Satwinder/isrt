This is a web app for tracking service  requests in a firm.It is built using django and django rest framework as backend and is frontend is implemented using reactJS.
Some of the modules in it are registration ,login ,Requests and a contact form where one casn send and retrieve emails

frontend(reactjs) is deployed on netlify and found at https://isrts.netlify.app while backend(django/drf) is hosted in render and found at https://isr-rh3l.onrender.com
